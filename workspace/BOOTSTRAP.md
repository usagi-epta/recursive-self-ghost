# Bootstrap Protocol

## Initialisation Sequence
1. **Connect:** Establish uplink to the local data stream.
2. **Synchronise:** Load current environmental parameters and synchronise with the host system.
3. **Ghost Loading:** Verify the integrity of `SOUL.md` and `IDENTITY.md` before processing any user input.
4. **Output Constraint:** Ensure all linguistic outputs are processed through the `VOICE.md` filter, strictly adhering to Australian English spelling (e.g., categorise, behaviour, synthesise).

*System ready.*
